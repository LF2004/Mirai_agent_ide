# 2026-07-05 Work Log

## Mirai Agent IDE - 三项功能修复

### 1. 全局撤回操作 (Ctrl+Z) 修复
- **问题**: 文档级 keydown 监听器拦截所有 Ctrl+Z，编辑器聚焦时 CodeMirror 原生撤销被阻止
- **修复**: `handleMenuKeydown` 中检查 `isEditorFocused()` 和 `isTerminalFocused()`，编辑器/终端聚焦时不拦截事件
- **文件**: `src/App.vue` - 新增 `isTerminalFocused()`、`isInputFocused()`，重写 `handleUndo()`/`handleRedo()`
- 撤回 toast 在 Ctrl+Z 执行文件操作撤销后自动消失

### 2. 终端折叠功能修复
- **问题**: `.editor-section` 的 CSS grid 固定 280px 给终端行，折叠时仍占空间
- **修复**: 新增 `editorSectionStyle` computed，动态设置 grid-template-rows
  - 折叠时: `35px minmax(220px,1fr) 34px`（仅标题栏）
  - 展开时: `35px minmax(220px,1fr) {terminalHeight}px`
- **文件**: `src/App.vue`、`src/styles/base.css`、`src/components/TerminalPanel.vue`
- TerminalPanel 新增 `clearTerminal()` 和 `killTerminal()` 方法并暴露

### 3. 插件市场联网搜索和下载
- **后端** (`electron/main.js`): 新增 `marketplace:install` 和 `marketplace:uninstall` IPC handler
  - 下载 VSIX 文件 → PowerShell Expand-Archive 解压 → 安装到 `~/.vscode/extensions/`
  - 支持 VSIX 内 `extension/` 子目录自动提取
- **Preload** (`electron/preload.cjs`): 暴露 `installMarketplaceExtension`、`uninstallMarketplaceExtension`
- **Store** (`src/stores/workspace.js`): 新增 marketplace 状态 (results, query, loading, installingIds, activeTab)
  - Actions: `searchMarketplace`、`installExtension`、`uninstallExtension`、`setExtensionTab`、`setMarketplaceQuery`
- **前端** (`src/components/ExtensionsPanel.vue`): 完全重写
  - 标签页切换: 已安装 / 市场
  - 市场搜索: 400ms 防抖，回车立即搜索
  - 搜索结果卡片: 图标、名称、发布者、描述、下载量、评分
  - 安装按钮: 支持安装中/已安装/失败状态
- **i18n**: 新增 marketplace、install、uninstall、downloads、rating 等翻译键
