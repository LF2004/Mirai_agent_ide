## Agent 核心实现

### Bug 修复
- `TerminalPanel.vue`: 修复 xterm.js theme 使用 CSS 变量字符串的 bug（xterm 不解析 `var(--xxx)`），改为 `getComputedStyle` 读取实际颜色值

### Agent 后端 (`electron/services/agent-service.js`)
- **AgentService 类**: 核心 Agent 引擎，支持 OpenAI 兼容 API
- **LLM 客户端**: 流式 SSE 调用 `chat/completions`，支持 function calling
- **工具引擎**: 8 个内置工具 — read_file, write_file, list_files, search_files, run_command, create_file, delete_file, get_workspace_info

- **模式支持**: agent (全功能) / ask (只读问答) / plan (只读+计划) / debug / multitask
- **安全限制**: plan/ask 模式禁用写工具；shell 命令 30s 超时；路径校验在 workspace 内
